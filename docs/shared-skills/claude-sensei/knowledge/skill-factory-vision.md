# Skill Factory — Vision

## Concept

Nymyka as a **skill factory**: building specialized Claude skills for domain experts and enterprises.

## Flow

1. Expert in domain X (doctor, lawyer, engineer, trader...)
2. Amaru protocol connects expert with Nymyka team
3. Nymyka builds personalized skill (calibrated with expert's knowledge)
4. Expert uses their skill daily in Claude
5. Revenue model: combo (consulting fee + subscription + marketplace cut)

## Target Markets

| Segment | Que necesitan | Ejemplo |
|---------|--------------|---------|
| **Individuals** | AI-augmented workflow personal | Abogado con skill juridico calibrado |
| **Enterprises** | Custom skill ecosystems para equipos | Soul-desk model (multi-agent customer service) |
| **Consultancies** | White-label skill development | Partners que revenden skills Nymyka |

## HERMES Role

- **Onboarding**: communication protocol during skill creation
- **Runtime**: inter-skill communication between different experts' skills
- **Governance**: Dojo manages skill lifecycle, evolution, quality
